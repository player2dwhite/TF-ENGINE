What is TF-Engine?

TF-Engine is a pure CSS design system created for Fandom and MediaWiki wikis. It provides reusable cards, panels, tabs and UI components.

Why use TF-Engine?

It keeps wiki pages consistent while allowing editors to create modern layouts without writing custom CSS every time.
