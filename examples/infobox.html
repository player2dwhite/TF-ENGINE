Article Example

This page demonstrates how the TF-Engine infobox works inside a MediaWiki article. The infobox is designed to behave like traditional Fandom character, location and item infoboxes.

The article content continues normally while the infobox remains separated from the main text using the component styling.

Overview

TF-Engine provides reusable wiki components that can be combined with normal wikitext content. Editors can create consistent layouts without manually writing custom CSS for every page.

History

Additional article sections can continue below the infobox, just like a normal Fandom wiki page.
